Welcome to the Richmond Airdrop

The Richmond Airdrop is a community-run effort created to provide support during a time of widespread uncertainty.

If you are looking for assistance or support with everyday tasks, you can request access to the Richmond Airdrop Signal group using the buttons below.

To help ensure your request is seen promptly, please message all three admins below. This reduces delays if one admin is unavailable.

Request Access

Message Admin 1 on Signal Message Admin 2 on Signal Message Admin 3 on Signal

Please send the same message to each admin. This helps ensure you can be added within 24 hours.

Message to send:

Please add me to this Richmond Airdrop Signal group
